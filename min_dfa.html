Introduction

Convert simple regular expressions to minimum deterministic finite automaton. (Regex => NFA => DFA => Min-DFA)

Supported raw regex grammars

  • r = (s)
  • r = st
  • r = s|t
  • r = s*
  • r = s+
  • r = s?
  • r = [stu]
  • r = A-Z
  • r = a-z
  • r = 0-9 (note that we only support these 3 character groups exactly)
  • r = \_ (supports 1 char-long escaped characters like \n or \+)
  • r = ϵ
    (Copy this character to input if needed)

Supported parsed regex grammars

  • r = (s)
  • r = st
  • r = s|t
  • r = s*
  • r = s+
  • r = s?
  • r = ϵ
    (Copy this character to input if needed)

Examples

  • (a|b)*
  • (a*|b*)*
  • ((ϵ|a)b*)*
  • (a|b)*abb(a|b)*
